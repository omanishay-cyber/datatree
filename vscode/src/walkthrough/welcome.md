# Welcome to Mneme

Mneme is a local-first persistent memory and code graph for AI coding agents.

What it gives you, out of the box:

- A **code graph** of your workspace — every function, class, and call edge.
- A **decision ledger** that captures "why" alongside "what", so context survives between sessions.
- **Drift scanners** that spot regressions the moment you save.
- An **MCP server** so Copilot Chat, Claude Code, Cursor, and other agents can recall facts instantly.

Everything runs on your machine. Nothing leaves it.
