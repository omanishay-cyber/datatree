# Try a recall

Press `Ctrl+K M R` (or run **Mneme: Recall** from the command palette).

Type a query like:

- `compaction recovery`
- `auth flow`
- `why did we switch to sqlite`

Mneme will return ranked hits across **code**, **decisions**, and **docs**.
Click one to jump straight to the source location.

Recalls are typically **sub-100 ms** once the index is warm.
