# Stay in sync

Once the daemon is running, mneme feeds VS Code live:

- **Hover** any symbol in any language to see its blast radius, recent decisions, and tests.
- **CodeLens** above functions shows `N callers | M tests | K edits`.
- **Diagnostics** from drift scanners appear in the Problems panel and as inline squiggles — refreshed every 15 seconds and on every save.
- **Toasts** for critical findings, daemon events, and stale indexes.

If you want to slow it down, the **Mneme** settings panel has knobs for every surface:
hover, code lens, drift poll interval, god-node count, log level, and more.

You're set. Happy coding.
